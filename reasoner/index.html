Reasoner

Bases: ABC

Base class (or rather interface) for reasoners.

Reasoners are specialized inference engines / "modules" of the (main) solver. They are queried by the solver's propagation method one by one, to perform specialized propagation / inference. Each of them processes newly accumulated events (bound updates), including those resulting from inference / propagation by other reasoners, until nothing new can be inferred.

When a reasoner encounters a conflict, it returns an initial / base / "starting point" explanation.
